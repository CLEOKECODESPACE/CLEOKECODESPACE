## ğŸš¦ Getting Started

### Prerequisites
- Node.js 18+ and npm/yarn
- PostgreSQL database
- Meta Developer Account with App ID
- WhatsApp Business Account

### Quick Start

1. **Clone and setup**:
```bash
git clone <repository-url>
cd "META PULSE"
```

2. **Backend setup**:
```bash
cd backend
npm install
cp .env.example .env
# Configure your environment variables
npm run dev
```

3. **Frontend setup**:
```bash
cd frontend
npm install
npm run dev
```

4. **WhatsApp Bot setup**:
```bash
cd whatsapp-bot
npm install
cp .env.example .env
# Configure WhatsApp Business API credentials
npm run dev
```
